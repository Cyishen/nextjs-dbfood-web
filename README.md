## Getting Started

First, run the development server:

```bash
npm run dev
```

```
yarn add prisma @prisma/client
```

##### docker folder install
```
cd src/docker
docker compose up
```

##### add prisma migrate
```
npx prisma migrate dev
```

##### see prisma studio
```
npx prisma studio
```
